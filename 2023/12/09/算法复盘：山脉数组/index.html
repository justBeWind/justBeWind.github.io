算法复盘：山脉数组

发表于2023-12-09|更新于2023-12-14

题目

题目链接：LCR 069. 山脉数组的峰顶索引 - 力扣（LeetCode）

符合下列属性的数组 arr 称为 山峰数组（山脉数组） ：

  • arr.length >= 3

  • 存在

    使得：

    • arr[0] < arr[1] < ... arr[i-1] < arr[i]
    • arr[i] > arr[i+1] > ... > arr[arr.length - 1]

给定由整数组成的山峰数组 arr ，返回任何满足 arr[0] < arr[1] < ... arr[i - 1] < arr[i] > arr[i + 1] > ... > arr[arr.length - 1] 的下标 i ，即山峰顶部。

要求：请用两种方法解决这道题目

方法一：暴力遍历法

可以观察发现，从左往右查找峰顶的过程中 数值是不断上升的，而过了峰顶后，数值就出现了下降，因此只要找到第一个 满足它的后一位比自己小的，就是峰顶

方法二：二分查找，可以看到在峰顶位置的左手边是单调递增的，右边是单调递增的，这种有序性的规律 可以用到二分查找

所以我们只需要从左往右找到第一个满足它的后一位比自己小的，就是峰顶，因此也可以利用二分查找。是个左边界查找问题

复盘：主要是没想到单调性 还能结合二分查找的特点，然后巧妙利用arr[mid] > arr[mid + 1] 这个不等式
